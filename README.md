# weapon_detection_model_tiny
tranfer learning yolov4-tiny model to detect 4 different classes of weapons

## classes
```
Handgun
Knife
Rifle
Sword
```

## notebook workflow
1. install darknet
2. use pre-trained yolov4-tiny on a random image
3. train yolov4-tiny on custom dataset
4. test and evaluate
